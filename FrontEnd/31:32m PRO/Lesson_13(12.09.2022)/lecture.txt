ТЕМЫ УРОКА
    - DOM 
    - ссылание на элементы через команды:
        - .querySelector() // метод, возвращающий элемент по селектору
        - .querySelectorAll() // метод, возвращающий итерируемый объект (NodeList) по селектору
        - .getElementsByClassName() // метод, возвращающий итерируемый объект (HTMLCollection) по селектору
    - NodeList ("не массив") // Итерируемый объект. В НодЛисте не работают методы Массивов. Работает только ForEach
    - .innerText() - изменение текста
    - .addEventListener() - позволяет повесить событие на параграф и при этом выполняется callback функция
    - .style как метод изменения стилей через js
